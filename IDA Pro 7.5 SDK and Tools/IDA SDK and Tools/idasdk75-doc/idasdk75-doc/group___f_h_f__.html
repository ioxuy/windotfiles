◆ FHF_VERIFY

#define FHF_VERIFY   0x0001

verify that the value fits into WIDTH

bits. If this property is not set we just truncate the value.

◆ FHF_CODE

#define FHF_CODE   0x0002

verify that ITEM_EA in std_apply() points

to an instruction.

◆ FHF_FORCE_CODE

#define FHF_FORCE_CODE   0x0004

if ITEM_EA in std_apply() points to an

unknown item, then convert it to code. this property is valid only with FHF_CODE.
