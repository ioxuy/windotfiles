◆ COLOR_ESC

#define COLOR_ESC   '\3'

Escape character (Quote next character).

◆ COLOR_INV

#define COLOR_INV   '\4'

Escape character (Inverse foreground and background colors).

This escape character has no corresponding COLOR_OFF. Its action continues until the next COLOR_INV or end of line.
